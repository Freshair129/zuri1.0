## 2. TAM / SAM / SOM

### TAM (Total Addressable Market)
**SME ภาคบริการทั้งหมดที่ใช้ social media ขายของ**
- Universe: ~450K-550K businesses
- Avg. SaaS spend: ฿500-1,500/mo
- **TAM = ~$100-300M USD/year**

### SAM (Serviceable Available Market)
**SME 3-50 คน, revenue ฿1M-50M, ใช้ FB+LINE actively**
- Universe: ~150K-200K businesses
- Avg. SaaS spend: ฿800-2,000/mo
- **SAM = ~$45-140M USD/year**

### SOM (Serviceable Obtainable Market — 3 ปี)
**Culinary schools + Beauty + Fitness ใน กทม.**
- Bangkok target verticals: ~25K-35K businesses
- Realistic penetration 3 ปี: 2-5% = 500-1,750 customers
- At avg. ฿1,500/mo:
- **SOM = $270K-$950K ARR (Year 3)**
- Aggressive (7-10%, ขยาย เชียงใหม่/ภูเก็ต): **$1.5-2.5M ARR**

### 3.2 Key Profiles

**R-CRM (Readyplanet)**
- จุดแข็ง: ตลาดไทยมานาน, 100K+ ลูกค้า Readyplanet ecosystem, ราคาถูก
- จุดอ่อน: CRM basic, ไม่มี POS, ไม่มี unified inbox, ไม่มี AI, UI ค่อนข้างเก่า

**ZWIZ.AI**
- จุดแข็ง: LINE chatbot อันดับ 1 ของไทย, Thai NLP, 50K+ LINE OA connected
- จุดอ่อน: LINE-first (FB รอง), CRM lightweight, ไม่มี POS, ไม่มี vertical features

**Oho Chat**
- จุดแข็ง: Unified inbox ครบ (LINE+FB+IG+Web), UI สวย, ราคาเข้าถึง
- จุดอ่อน: ไม่ใช่ CRM เต็มรูปแบบ, ไม่มี POS, ไม่มี AI generative, ไม่มี vertical features
- **⚠️ คู่แข่งที่ต้องจับตา — ใกล้เคียง Zuri Inbox มากที่สุด**

**FoodStory POS**
- จุดแข็ง: POS สำหรับร้านอาหาร, ราคาถูก, มี QR ordering, 20K+ restaurants
- จุดอ่อน: ไม่มี CRM, ไม่มี inbox, inventory basic มาก

**Choco CRM** ⭐ NEW
- จุดแข็ง: Thai CRM + POS + Loyalty/Membership ในตัวเดียว, LINE OA integration, 10K+ businesses (Choco Card ecosystem), ราคาเข้าถึงได้
- จุดอ่อน: ไม่มี unified inbox (FB+LINE), ไม่มี Ads analytics จริง, ไม่มี Kitchen Ops/Enrollment, ไม่มี AI, POS basic กว่า FoodStory
- **⚠️ คู่แข่งที่ใกล้เคียงที่สุดในไทย — มีทั้ง CRM + POS + Loyalty**
- **vs R-CRM**: R-CRM เก่งเรื่อง sales pipeline/lead mgmt; Choco เก่งเรื่อง B2C loyalty/membership

**Kintone (Cybozu)** ⭐ NEW
- จุดแข็ง: No-code platform จากญี่ปุ่น, workflow automation ดีมาก, Thai UI, ยืดหยุ่นสร้างอะไรก็ได้, เป็นที่นิยมในบริษัทญี่ปุ่นในไทย
- จุดอ่อน: ต้อง build ทุกอย่างเอง (ไม่มี CRM/POS/Inbox สำเร็จรูป), ไม่มี LINE/FB integration, ราคา ~฿840/user (10 คน = ฿8,400), learning curve สูง
- **ไม่ใช่คู่แข่งตรง** — เป็น platform ไม่ใช่ vertical SaaS แต่ลูกค้าอาจเลือก Kintone แทนเพราะ "สร้างเองได้ทุกอย่าง"

**SuperPOS** ⭐ NEW
- จุดแข็ง: POS ไทยราคาถูก (฿800-5,000), ใช้งานง่าย, มี KDS basic
- จุดอ่อน: POS only, ไม่มี CRM/Inbox/AI/Enrollment/Kitchen Ops, ecosystem เล็ก, เสี่ยงเลิกให้บริการ
- **คู่แข่งระดับ FoodStory** — เฉพาะ POS ไม่ใช่ all-in-one
